### 简单的节点脚本
简单的节点脚本，可以完成一些日常小任务，文件处理，网络抓取等。更多功能完善中...
```shell
npm i
npm run dev
```
 _编辑器：ts+pixijs，执行器：rust_ 

##### 使用
编辑器导出 .sgs 文件，执行 `sgs-vm my-script.sgs`
 **sgs-vm** 默认在应用目录下bin/
